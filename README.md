# nodeAndMySQL


The user will be prompted with a list of the available products, and the ID that goes along with them.  
The user will then select input the ID of the product they wish to buy.
The user will be prompted for an amount of the product they wish to buy.
The code will then check to see if there is enough product to sell to the user.
If there is the sale goes through, if not the code will tell the user that there is not enough product to sell them the whole amount they requested.
